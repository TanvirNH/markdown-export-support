Frequently Asked Questions

What file formats can I export to? ‚ñº

Markdown Export supports exporting your documents to PDF, Word (.docx), and PNG image formats. Simply write your markdown and click the export button for your desired format.

Can I open and edit existing markdown files? ‚ñº

Yes! Use the "Open .md" button in the sidebar to open any existing markdown file. The content will be loaded into the editor and you can make changes, then export in your desired format.

How do I adjust text formatting in exports? ‚ñº

Use the control panel on the left sidebar. You can change the font face, adjust text size (8-16pt), and control border strength. These settings affect how your export looks.

Can I use images in my markdown? ‚ñº

Yes! Use markdown image syntax: ![Alt text](image-url). You can use external image URLs. The app will render them in the preview and include them in your exports.

Is there a dark mode? ‚ñº

Absolutely! Click the dark mode toggle button (‚óë) in the top right of the sidebar to switch between light and dark themes.

Does the app save my documents automatically? ‚ñº

Markdown Export doesn't auto-save, but you can export your work anytime. We recommend using the "Open .md" button to load previously saved markdown files from your computer.

Can I create multi-page documents? ‚ñº

Yes! The preview shows page breaks automatically. You can see where page 1 ends and page 2 begins, so you can organize your content accordingly before exporting.

What markdown syntax is supported? ‚ñº

Markdown Export supports standard markdown including headers (# ## ###), bold (**text**), italic (*text*), lists, code blocks, tables, links, images, blockquotes, and more.
